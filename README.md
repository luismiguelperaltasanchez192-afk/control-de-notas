# Sistema de Control y Promedio de Calificaciones en C++

Este repositorio contiene una solución en C++ desarrollada para la gestión automatizada de calificaciones estudiantiles, cumpliendo con los requerimientos prácticos de la asignatura** Lógica de Programación** .

## 📝 Descripción del Problema
El programa procesa dinámicamente el registro de una cantidad* n* de estudiantes mediante estructuras repetitivas(bucles).Para cada estudiante, el sistema captura su nombre completo y cuatro calificaciones parciales.De forma automática, calcula el promedio final y determina el estatus académico del alumno bajo el siguiente criterio :
***Aprobado : **Promedio mayor o igual a 70.0.
* **Reprobado : **Promedio menor a 70.0.

El despliegue final de la información se genera directamente en la consola del sistema, utilizando tabulaciones estandarizadas(`\t`) para estructurar los datos visualmente en forma de tabla.

	-- -

	## 💻 Requisitos Implementados
	* **Estructuras de Control : **Uso de bucles interactivos para la captura limpia de datos sin importar el volumen de estudiantes.
	* **Formato Tabular : **Aplicación de espaciado y tabulación en la salida de consola para garantizar la legibilidad.
	* **Modularidad : **Uso de estructuras de datos nativas de C++ (`struct` y `vector`) para un manejo eficiente de la memoria y la información.
